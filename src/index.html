Introduktion


Den här uppgfiten handlar om att man ska kunna skapa och automatisera en webbsida med hjälp av Node.js och Parcel och kunna automatiskt versionhantera den med hjälp av Git och Github. Webbplatsen kommer då att vara tillgänglig på en webbhost som till exempel Netlify.





Frågeställningar


De problem som ska lösas med den här demo-webbplatsen är:

  1. Automatiseringen av:
    • Minifiering av CSS.
    • Transpilering och minifiering av JavaScript.
    • Minifiering av HTML.
    • Optimering av bilder - skall ske automatiskt vid uppdatering av bilder.
    • Bakåtkompabilitet i genererade filer, så att exempelvis JavaScript transpileras så att dessa fungerar i så många webbläsare som möjligt.
    • Kopiering av arbetsfiler till mapp för publicering med optimerade filer.
  2. Automatisk publicering vid versionhanteringen av koden med hjälp av Github och publiceringen till Netlify eller liknande webbhost.
